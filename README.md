# zshrc
- .zshrc: dotfile for individual user accounts, simply sources /etc/zshrc
- zshrc: to be placed in `/etc/zshrc`

# vimrc
- .vimrc: dotfile for individual user accounts, disables mouse mode entirely
- vimrc.local: to be placed in `/etc/vim/vimrc.local`, adapted from [amix/vimrc](https://github.com/amix/vimrc)

# tmuxrc
- tmux.conf: vim mode for tmux
